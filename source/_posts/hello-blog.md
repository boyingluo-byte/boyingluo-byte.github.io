# 博客搭建完成！

经过一番折腾，终于把博客搭建好了。技术栈如下：

## 技术选型

- **框架**: Hexo（静态站点生成器）
- **主题**: anzhiyu（安知鱼主题，非常好看）
- **部署**: GitHub Pages（免费、稳定）

## 搭建步骤

1. 安装 Node.js 和 Git
2. 安装 Hexo CLI：`npm install -g hexo-cli`
3. 初始化项目：`hexo init blog`
4. 安装 anzhiyu 主题：`npm install hexo-theme-anzhiyu`
5. 配置主题并部署
